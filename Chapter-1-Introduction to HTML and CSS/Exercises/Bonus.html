HTML & CSS Reference Guide

Basic HTML Tags and CSS Properties

HTML Tags

1. <h1> - Main Heading

This tag is used for the main heading of the document.

<h1> Welcome to my Website </h1>

2. <p> - Paragraph

Define a paragraph of text.

<p> This is a simple paragraph explanation of my website.

3. <link> - Link External Resources

Used to link stylesheet and other resources.

<link rel="stylesheet" href="style.css">

4. <img> - Image

Embed a image into the website 

<img src="https://www.example.com" alt="Example website" >

5. <ul> <li> - Unordered List

Create an unordered list with list items.

<ul>
                <li>Apples</li>
                <li>Bananas</li>
                <li>Cherries</li>
            </ul>

6. <ul> <li> - Ordered List

Create an ordered.

<ul>
                <li>Wake up</li>
                <li>Brush teeth</li>
                <li>Have breakfast</li>
            </ul>

7. <form> - Form

Define an HTML form for user input.

<form action= "submit.php">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name">
                <input type="submit" value="submit">
                </form>

8. <footer> - Footer

Define footer at the bottom of the document.

<footer>
                        <p>©Haya waseem</p>
                    </footer>

9. <nav> - Navigation

Defines navigation links.

<nav>
                            <a href="#home">Home</a>
                            <a href="#about">About</a>
                          </nav>

10. <input>- Input

Creates a variety of input controls depending on the type.

<input type= "text" placeholder="Enter name">

CSS Properties

1. Background-color

Sets the background color of an element.

body {background-color: lightblue;}

2. Font-family

Sets the font family for an element.

h1 {font-family: Arial, sans-serif;}

3. Color

Sets the text color of an element.

p {color: blue;}

4. Margin

Sets the margin space around an element.

div {margin: 10px;}

5. Padding

Sets the padding inside an element.

section { padding: 20px;}

6. Border

Defines a border around an element.

div {border: 2px solid black;}

7. Text-align

Aligns the text inside an element.

h1 {text-align: center;}

8. Position

Specifies the positioning method of an element.

div {position: relative;
                top: 20px;
                left: 30px;}
