HTML & CSS Reference Guide

Basic HTML Tags and CSS Properties

HTML Tags

<h1> - Main Heading

This tag is used for the main heading of the document.

<h1> Welcome to my Website </h1>

<p> - Paragraph

Define a paragraph of text.

<p> This is a simple paragraph explanation of my website.

<a> - Anchor (Hyperlink)

Create a Hyperlink.

<a href="https://www.wikipedia.org" >Visit Example Site</a>

<img> - Image

Embed a image into the website 

<img src="https://www.example.com" alt="Example website" >

<ul> <li> - Unordered List

Create an unordered list with list items.

<ul>
                <li>Apples</li>
                <li>Bananas</li>
                <li>Cherries</li>
            </ul>

<ul> <li> - Ordered List

Create an ordered.

<ul>
                <li>Wake up</li>
                <li>Brush teeth</li>
                <li>Have breakfast</li>
            </ul>

<form> - Form

Define an HTML form for user input.

<form action= "submit.php">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name">
                <input type="submit" value="submit">
                </form>

<footer> - Footer

Define footer at the bottom of the document.

<footer>
                        <p>©Haya waseem</p>
                    </footer>

<nav> - Navigation

Defines navigation links.

<nav>
                            <a href="#home">Home</a>
                            <a href="#about">About</a>
                          </nav>

CSS Properties

Background-color

Sets the background color of an element.

body {
  background-color: lightblue;
}

Font-family

Sets the font family for an element.

h1 {
  font-family: Arial, sans-serif;
}

Color

Sets the text color of an element.

p {
  color: blue;
}

Margin

Sets the margin space around an element.

div {
  margin: 10px;
}

Padding

Sets the padding inside an element.

section {
  padding: 20px;
}

Border

Defines a border around an element.

div {
  border: 2px solid black;
}

Text-align

Aligns the text inside an element.

h1 {
  text-align: center;
}

Display

Specifies how an element is displayed (block, inline, etc.).

span {
  display: inline;
}

Position

Specifies the positioning method of an element.

div {
  position: relative;
  top: 20px;
  left: 30px;
}
