HTML & CSS Reference Guide

Basic HTML Tags and CSS Properties

HTML Tags

<h1> - Main Heading

This tag is used for the main heading of the document.

<h1> Welcome to my Website </h1>

<p> - Paragraph

Define a paragraph of text.

<p> This is a simple paragraph explanation of my website.

<a> - Anchor (Hyperlink)

Create a Hyperlink.

<a href="https://www.wikipedia.org" >Visit Example Site</a>

<img> - Image

Embed a image into the website 

<img src="https://www.istockphoto.com/photo/autumn-on-lake-gosau-in-salzkammergut-austria-gm2166773378-586863679?utm_campaign=srp_photos_top&utm_content=https%3A%2F%2Funsplash.com%2Fs%2Fphotos%2Flandscape&utm_medium=affiliate&utm_source=unsplash&utm_term=landscape%3A%3A%3A" alt="A beautiful landscape" >

<ul> <li> - Unordered List

Create an unordered list with list items.

<ul>
                <li>Apples</li>
                <li>Bananas</li>
                <li>Cherries</li>
            </ul>

<ul> <li> - Ordered List

Create an ordered.

<ul>
                <li>Wake up</li>
                <li>Brush teeth</li>
                <li>Have breakfast</li>
            </ul>

<form> - Form

Define an HTML form for user input.

<form action= "submit.php">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name">
                <input type="submit" value="submit">
                </form>

<footer> - Footer

Define footer at the bottom of the document.

<footer>
                        <p>©Haya waseem</p>
                    </footer>

<nav> - Navigation

Defines navigation links.

<nav>
                            <a href="#home">Home</a>
                            <a href="#about">About</a>
                          </nav>
